# 3D Model ফোল্ডার

এখনো এখানে কোনো ফাইল নেই। এই ফোল্ডার ভবিষ্যতে real .glb fixture/product model রাখার জন্য।

## কী ডাউনলোড করতে হবে (প্রথম পছন্দ)

Sketchfab → **CC0 - Coat Rack 3** (by plaggy)
https://sketchfab.com/3d-models/cc0-coat-rack-3-0d27167d172443a895670a41375f44db

CC0 1.0 Universal license, মাত্র ১৬৬ polygon (অত্যন্ত হালকা, mobile-friendly), PBR texture সহ, glTF format-এ direct download করা যায়।

## কীভাবে ডাউনলোড করবে (ফোনে)

1. উপরের লিংকে যাও, Sketchfab account দিয়ে log in করো (ফ্রি, না থাকলে বানাতে হবে)।
2. "Download 3D Model" বাটনে — format হিসেবে **glTF** বাছো।
3. ডাউনলোড হওয়া ফাইলটা এই ফোল্ডারে রাখো এই নামে: `coat-rack.glb`

## বিকল্প

যদি ভিন্ন কিছু পছন্দ হয়, Sketchfab-এ "downloadable" + "CC0" filter দিয়ে খুঁজলে আরো option পাবে: https://sketchfab.com/tags/clothes-rack — শুধু license CC0 বা CC-BY আছে কিনা চেক করে নিও (CC-BY-SA হলে attribution লাগবে)।
